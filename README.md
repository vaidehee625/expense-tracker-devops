# Expense Tracker DevOps Project

## Overview

A full-stack Expense Tracker application built using Spring Boot, MongoDB, Docker, Jenkins, and AWS.

## Features

- Add Expenses
- View Expenses
- Delete Expenses
- MongoDB Atlas Integration
- Docker Deployment
- Jenkins CI/CD Pipeline

## Technology Stack

- Java Spring Boot
- MongoDB Atlas
- Docker
- Jenkins
- AWS EC2
